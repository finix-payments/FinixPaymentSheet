Finix Payment Sheet

iOS card tokenization UI. Provides a customizable UI + optional presentation to tokenize a credit card.

Building

Tool dependencies

This project uses the following:

  • Mint
  • SwiftFormat
  • xcodegen

Please make sure to also have Xcode installed. The latest can be found via https://xcodereleases.com To set up the tools, the easist way to install mint is with homebrew via:

brew install mint

Once mint is installed install the dependencies with:

make bootstrap

Now the dependencies are installed.

Build the xcodeproj

Now generate the .xcworkspace with:

make update

**NOTE: this is the default target so you can run make by itself.

This will generate the project file that you can browse with Xcode with:

open FinixPaymentSheet.xcodeproj

Build XCFramework

This project builds an XCFramework for simple compabitility with simulator and device. To build:

make xcframework

Once it finishes the framework will be in build/ named FinixPaymentSheet.xcframework.

Building a Release

make artifacts

This will build the XCframework and copy PaymentSheetDemo with a working .xcodeproj as well as a CocoaPods podspec for the XCframework. It can then be checked into the binary repo & tagged with a release version. Make sure the tag matches the version in Podspec.

Demo App

There is a demo app that can be run via Xcode. The build target is PaymentSheetDemo and it runs on simulator and device.
